# Security

If you believe you’ve found a security vulnerability, please report it responsibly.

## How to report

- **Preferred:** Open a [private security advisory](https://github.com/Haroon966/Cove/security/advisories/new) on GitHub so we can discuss and fix it before any public disclosure.
- Alternatively, describe the issue in a private way (e.g. email if you have a maintainer contact) rather than in a public issue.

## What to include

- Steps to reproduce.
- Impact (what an attacker could do).
- Any suggested fix or mitigation, if you have one.

We’ll do our best to respond quickly and will credit you for the report if you’re happy to be named (after the issue is fixed).
